Detailed Description

C++ representation of a telemetry data sample.

This struct provides a convenient C++ interface for telemetry data, avoiding Protobuf boilerplate in application code. Use ProtoAdapter to convert to/from Protobuf messages for serialization.

Note
This is not a Protobuf message - use ProtoAdapter for serialization
