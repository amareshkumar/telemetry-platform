\doxysubsection{Detailed Description}
Redis publisher for Telemetry\+Hub-\/\+Telemetry\+Task\+Processor integration. 

Publishes Telemetry\+Sample data as tasks to Redis for async processing by Telemetry\+Task\+Processor workers.

Integration Architecture\+:

Telemetry\+Hub Device → \doxylink{classtelemetryhub_1_1gateway_1_1_telemetry_queue}{Telemetry\+Queue} → \doxylink{classtelemetryhub_1_1gateway_1_1_gateway_core}{Gateway\+Core} → \doxylink{classtelemetryhub_1_1gateway_1_1_redis_publisher}{Redis\+Publisher} ↓ RPUSH Redis (task queue) ↓ BLPOP Telemetry\+Task\+Processor Workers
