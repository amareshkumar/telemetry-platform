\doxysubsection{Detailed Description}
Simple thread pool for processing telemetry samples. 

Features\+:
\begin{DoxyItemize}
\item Fixed number of worker threads
\item Job queue with FIFO processing
\item \doxylink{structtelemetryhub_1_1gateway_1_1_thread_pool_1_1_metrics}{Metrics}\+: jobs processed, average processing time
\item Graceful shutdown with job completion
\end{DoxyItemize}

Design considerations\+:
\begin{DoxyItemize}
\item Reduces thread creation overhead for high-\/frequency tasks
\item Limits concurrency for bounded resource usage
\item Better CPU utilization through worker reuse
\item Minimal queue latency for telemetry processing workload 
\end{DoxyItemize}
