\doxysubsection{Detailed Description}
C++ representation of a telemetry data sample. 

This struct provides a convenient C++ interface for telemetry data, avoiding Protobuf boilerplate in application code. Use \doxylink{classtelemetry_1_1_proto_adapter}{Proto\+Adapter} to convert to/from Protobuf messages for serialization.
