\doxysubsection{Detailed Description}
Represents a single unit of work with priority and metadata. 

A task encapsulates\+:
\begin{DoxyItemize}
\item {\bfseries{Priority}}\+: For scheduling order
\item {\bfseries{Timestamp}}\+: For FIFO ordering within same priority
\item {\bfseries{Payload}}\+: Arbitrary JSON data for processing
\item {\bfseries{ID}}\+: Unique identifier for tracking
\end{DoxyItemize}

{\bfseries{Design Decisions}}\+:
\begin{DoxyItemize}
\item JSON payload\+: Flexible schema-\/less data (vs rigid structs)
\item Timestamp\+: std\+::chrono for precision (microseconds)
\item ID\+: String UUID for distributed systems
\end{DoxyItemize}
