\doxysubsection{Detailed Description}
Comparator for priority queue ordering. 

Orders tasks by\+:
\begin{DoxyEnumerate}
\item Priority (lower enum value = higher priority)
\item Timestamp (earlier = higher priority for same priority level)
\end{DoxyEnumerate}

This ensures\+:
\begin{DoxyItemize}
\item HIGH priority processed before MEDIUM/\+LOW
\item FIFO ordering within same priority
\end{DoxyItemize}
